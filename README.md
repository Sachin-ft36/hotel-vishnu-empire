# REGALIA — A Royal Hospitality Collection

REGALIA is an ultra-luxury hospitality platform featuring cinematic hotel presentations, heritage destinations, and bespoke experiences.

## Tech Stack

- **Framework**: React + Vite
- **Styling**: Tailwind CSS
- **Animations**: CSS3 Transitions & Keyframes
- **Components**: Custom luxury design system

## Development

```bash
npm run dev
```

## Production Build

```bash
npm run build
```
